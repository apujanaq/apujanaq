### Projects

**[Customer Retention Model — Binary Classification](https://github.com/apujanaq/Proyecto_ML_Fidelizacion)**
Predicts which customers of an online retailer will repurchase in the next 3 months.
Pipeline over 1M+ transactions (UCI Online Retail II): feature engineering (RFM),
Logistic Regression / Random Forest / XGBoost + Optuna hyperparameter tuning.
AUC-ROC ~0.83. Output: prioritized list of 1,088 customers with repurchase probability
and RFM segment for the commercial team.

**[Statistical Analysis for Data Science](https://github.com/apujanaq/Evolve-Estadistica-Alejandro-Pujana)**
Four applied exercises on real data (Loan Approval 2025, 50K records + synthetic data):
descriptive EDA, multiple linear regression with Scikit-learn, OLS implemented from
scratch with NumPy (β = (X'X)⁻¹X'y, validated against sklearn with Δ = 0),
and time series decomposition with ADF and Jarque-Bera tests.
